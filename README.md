# ALX CSS Project

This is a frontend project for ALX, focused on applying CSS styling techniques to an HTML page based on a Figma design.

## Features

- Semantic HTML structure
- Responsive CSS layout
- Assets sourced from Figma
- Clean folder organization

## Technologies Used

- HTML5
- CSS3
- Fonts: Source Sans Pro
- Images: PNG assets from Figma design

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. No server needed, this is a static project
